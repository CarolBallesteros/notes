Usar nombres que revelen las intenciones:
    el nombre de una variable, función o clase debe responder una seria de cuestiones basicas:
    debe indicar:
	¿Por qué existe?
	¿Qué hace?
	¿Como se usar?


Su un nombre requiere un comentario, significa que no revela su cometido:
    ejemplo:
	int d; // tiempo transcurrido en dias.

el nombre d no revela nada. No evoca una sensación de tiempo transcurrido, ni de dias. Debe elegir un nombre
que especifique lo que se mide y la unidad de dicha medida:
    ejemplo:
	int diasDesdeCreacion;
	int diasDesdeModificacion;
